Dashboard for Cálculo Precision application (Mobile App)

Key Features:
- Big top header with text "CÁLCULO PRECISION"
- Level progress section featuring:
  - A circular badge indicator showing "Level 2" and subtext "Estudante de Cálculo"
  - XP section showing "320 XP" in orange and "180 para próximo nível" text
  - A linear progress bar at 28% filled with bright yellow/green color
- A large search bar with a magnifying glass icon reading "O QUE VAMOS ESTUDAR HOJE?"
- Four massive, brutalist vertical navigation buttons with distinct vibrant background colors:
  - "FLASHCARDS" (Lime Green with lightning/card icon)
  - "MODO AULA" (Cyan/Teal with a teacher/presentation icon)
  - "CHAT IA" (Royal Blue with robot/chat icon)
  - "DESAFIO DO GUIDORIZZI" (Orange with a trophy/book icon)
- Bottom navigation bar containing four icons: Home (active), Search, Statistics, User Profile

Visual Style:
- Premium Neobrutalism UI style
- Intense, thick deep black borders (border width 4px+) on all interactive elements and containers
- Distinct, hard, dark 3D shadows (offset like 6px 6px 0px black) to make elements pop significantly
- Circuit board/tech grid subtle background texture in dark gray beneath all elements
- Vibrant, high-contrast flat colors (Orange, Lime Yellow, Cyan, Blue) against the dark background
- Bold, legible, uppercase typography for all buttons
- Symmetrical pill-shaped buttons with angled depth

Platform: Mobile (iOS/Android responsive size)
