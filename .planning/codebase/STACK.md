# Tech Stack - Guidorizzi App

## Core Framework
- **Runtime**: Node.js (ES Modules)
- **Frontend**: React 19 with Vite 7
- **Language**: TypeScript (`.tsx`) + JavaScript (`.js`)

## Styling & Animation
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion v12

## AI & Data
- **AI Backend**: Groq SDK (Llama 3.3 70B)
- **Frontend AI**: Google GenAI
- **Validation**: Zod v3

## Mathematics & Rendering
- **Math Engine**: KaTeX
- **Interactive Graphs**: Mafs
- **Markdown**: react-markdown (w/ remark-math, rehype-katex)

## Tools & Utilities
- **Icons**: Lucide React
- **HTTP**: Express v5 (Backend Bridge)
- **CORS/Security**: cors, helmet, express-rate-limit
- **Environment**: dotenv
